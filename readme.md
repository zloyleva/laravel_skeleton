## Laravel 5.7 Skeleton

### Installation:

1. Run 
    ```bash
    make composer_dep
    #install composer dependency
    ```
2. Set permissions on the project directory so that it is owned by your non-root user:
    ```bash
    sudo chown -R $USER:$USER $PWD
    ```
